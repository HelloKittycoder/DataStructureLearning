### 链表
```text
线性数据结构
1. 动态数组
2. 栈
3. 队列
底层依托静态数组；靠resize解决固定容量问题
```

```text
链表 真正的动态数据结构
最简单的动态数据结构
更深入的理解引用（或者指针）
更深入的理解递归
辅助组成其他数据结构
```

```text
链表 Linked List
数据存储再“节点”（Node）中
class Node {
    E e;
    Node next;
}
优点：真正的动态，不需要处理固定容量的问题
缺点：丧失了随机访问的能力
```

```text
数组和链表的对比
数组最好用于索引有语义的情况 scores[2]
最大的优点：支持快速查询

链表不适合用于索引有语义的情况
最大的优点：动态
```
